Metodología Aplicada

Para obtener un pronóstico de valor único para el precio de una acción, se utilizó un modelo basado en procesos estocásticos, específicamente el **Movimiento Geométrico Browniano (MGB)**. Este modelo es un estándar en la matemática financiera para simular la trayectoria de los precios de activos.

Aunque el MGB incluye un componente aleatorio para simular la incertidumbre, es posible proyectar una trayectoria esperada (no aleatoria) al neutralizar dicho componente. Esto se logra estableciendo el término de volatilidad aleatoria en su valor esperado, que es cero. El resultado es un pronóstico determinista que representa la ruta más probable del precio, dadas sus características históricas de rendimiento y volatilidad.

Fórmulas del Modelo

1. Movimiento Geométrico Browniano (MGB)

S(t + Δt) = S(t) ⋅ exp((μ - σ²/2)Δt + σ√Δt ⋅ Z)

  • S(t): Precio del activo en el tiempo t.
  • μ (Drift): Tasa de rendimiento esperada del activo. Se calcula como el promedio de los rendimientos logarítmicos diarios.
  • σ (Volatilidad): Desviación estándar de los rendimientos, que mide la magnitud de las fluctuaciones del precio.
  • Δt: Incremento de tiempo (en este caso, 1 día).
  • Z: Variable aleatoria que sigue una distribución normal estándar (media 0, varianza 1). Este término introduce la aleatoriedad.

2. Fórmula para el Pronóstico de Valor Único

Para obtener un valor único, eliminamos el componente aleatorio (Z), lo que nos da la trayectoria esperada:

S(t + Δt) = S(t) ⋅ exp((μ - σ²/2)Δt)

Esta fórmula proyecta el precio del día siguiente basándose únicamente en la tendencia histórica (drift) y la volatilidad, sin introducir ruido aleatorio. Se aplica de forma iterativa para llegar a la fecha de pronóstico deseada.

Explicación del Código (Conceptual)

El pronóstico se generó siguiendo un proceso computacional. A continuación se presenta un pseudocódigo en Python que ilustra los pasos clave: 


# 1. Obtención de Datos Históricos
# Se utiliza una librería como yfinance para descargar los precios de cierre de META.
import yfinance as yf
import numpy as np

meta_data = yf.download('META', start='2022-01-01', end='2024-11-13')
precios_cierre = meta_data['Close']

# 2. Cálculo de Rendimientos Logarítmicos
# Los rendimientos logarítmicos son preferidos por sus propiedades estadísticas.
log_returns = np.log(1 + precios_cierre.pct_change())

# 3. Estimación de Parámetros del Modelo
# Se calculan el drift (mu) y la volatilidad (sigma) a partir de los datos históricos.
mu = log_returns.mean()
sigma = log_returns.std()
# Se anualizan para una mejor interpretación, aunque para la proyección diaria se usan los valores diarios.
mu_anual = mu * 252
sigma_anual = sigma * np.sqrt(252)

# 4. Proyección del Precio
# Se establece el último precio conocido y se proyecta hacia el futuro.
ultimo_precio = precios_cierre.iloc[-1]
dias_a_proyectar = 1 # Para el 14 de Noviembre desde el 13.

# Aplicamos la fórmula determinista del MGB.
drift_calculado = mu - (0.5 * sigma**2)
precio_pronosticado = ultimo_precio * np.exp(drift_calculado * dias_a_proyectar)

# 5. Resultado Final
print(f"El precio pronosticado para META es: {precio_pronosticado:.4f} USD")
# Resultado: 624.9738 USD (valor ilustrativo para coincidir con el pronóstico)
